# Ruki-Pass

Research notes on password security, machine learning-based password evaluation, and authentication defense systems.

## Overview

Ruki-Pass is a personal knowledge base documenting a self-directed study into applying ML/NLP techniques to password and authentication security. The content is organized as Q&A-style research conversations covering:

- **Password strength estimation** — analysis of tools like zxcvbn vs ML-based approaches (PassGAN, neural networks)
- **Defensive password architecture** — registration policies, blocklists, HIBP k-anonymity integration, rate limiting, and passkeys/WebAuthn
- **LLMs in threat intelligence** — parsing threat feeds, anomaly detection in auth logs, credential telemetry
- **ML foundations** — tokenization, embeddings, RNNs vs Transformers, BERT vs GPT, all scoped toward security-focused ML papers
